# G4400_OpenCore
奔腾 G4400 黑苹果EFI

## 配置信息
| 名称 | 描述 |
| ---- | ---- |
| CPU | **奔腾 G4400 3.30GHz** |
| 主板 | **ASUS H110M-K** |
| 内存 | **8GB 2133MHz DDR4** |
| 硬盘 | **Teclast 128GB S550 Media** |
| 声卡 | **瑞昱 ALC887** |
| 网卡 | **瑞昱 RTL8111** |
| 显卡 | **NVIDIA GT640 1024MB** |
| 系统 | **Big Sur 11.4(20F71)** |
| OC版本 | **0.7.0**|

## 驱动情况
 - 声卡：正常（开机后默认声音输出设备为数码输出，调至内置扬声器输出即可听到声音）
 - 网卡：正常
 - 显卡：正常（GT640免驱）
 - 其他项目（如睡眠等）暂未测试

## 已知问题
 - PS2驱动无效，如使用 PS2 鼠标/键盘请自备 PS2=>USB转接线
